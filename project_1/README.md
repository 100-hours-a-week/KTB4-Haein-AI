# KTB 출석 계산기

Python으로 제작한 CLI 프로그램

## 프로젝트 소개

카카오테크 부트캠프의 출석 조건을 기준으로 제작한 출석 관리 프로그램

사용자가 날짜와 출석 정보를 입력하면 :

- 전체 출석률 계산
- 단위기간 출석률 계산
- 수료 가능 여부 판정
- 제적 위험 여부 판정

## 주요 기능

- 실제 수업일 계산
- 휴강 및 공휴일 제외
- 토요일 수업 반영
- 단위기간 자동 계산
- 출석률 자동 계산
- 지각/조퇴/외출 3회 누적 시 결석 1일 처리
- 전체 출석률 80% 미만 시 수료 불가능 판정
- 단위기간 출석률 50% 미만 시 제적 위험 판정

## 사용 기술

- Python
- datetime
- CLI(Command Line Interface)
- input()
- print()

## 실행 방법

```bash
python3 main.py
```
