# KTB 출석 계산기

Python으로 제작한 CLI 프로그램

## 프로젝트 소개

카카오테크 부트캠프의 출석 조건을 기준으로 제작한 출석 관리 프로그램

사용자가 날짜와 출석 정보를 입력하면 :

- 전체 출석률 계산
- 단위기간 출석률 계산
- 수료 가능 여부 판정
- 제적 위험 여부 판정

## 주요 기능

- 실제 수업일 계산
- 휴강 및 공휴일 제외
- 토요일 수업 반영
- 단위기간 자동 계산
- 출석률 자동 계산
- 지각/조퇴/외출 3회 누적 시 결석 1일 처리
- 전체 출석률 80% 미만 시 수료 불가능 판정
- 단위기간 출석률 50% 미만 시 제적 위험 판정

## 사용 기술

- Python
- datetime
- CLI(Command Line Interface)
- input()
- print()

## 실행 방법

```bash
python3 main.py
```

## 회고 
실제 사용 목적을 고려하여 프로그램을 설계하는 경험을 할 수 있었다. 처음에는 단순 출석 체크 프로그램을 생각했지만, 카카오테크 부트캠프의 실제 출석 조건을 반영하면 더 의미 있는 프로그램이 될 것 같아 방향을 변경하였다.

실제 수업일 수를 어떻게 정확하게 계산할 것인지에 대해 많은 고민이 있었다. 날짜 차이만 계산하는 것이 아니라, 주말과 휴강일, 공휴일을 제외해야 했고, 특정 기간까지는 토요일 수업이 존재한다는 조건도 추가해야 했다.

이를 해결하기 위해 Python의 datetime 모듈과 timedelta 를 활용하여 날짜를 하루씩 계산하는 방식을 적용하였다. 이 과정에서 날짜 데이터를 다루는 방법과 반복문을 활용한 실제 계산 로직을 배울 수 있었다. 

또한 지각, 조퇴, 외출 3회를 결석 1일로 처리하는 규칙과 단위기간 출석률 및 전체 출석률 조건을 구현하면서 조건문과 산술 연산의 활용 방법도 익힐 수 있었다.

이번 과제로부터 함수 분리, 날짜 처리, 조건 처리, 반복문 활용, GitHub 업로드 과정 등을 직접 경험할 수 있었고, 실제 사용 목적을 고려한 프로그램 설계의 중요성도 배울 수 있었다.
